# Git-Commands
# Push commands 
git init <br>
git status<br>
git add . (to push every element in your repo) or git add (specific file name)<br>
git status<br>
git commit -m "any comment"<br>
git log (it will show you history)<br>
git remote add origin (url of your repo)  or to change the origin  git set-url origin (url of your repo)<br>
git remote -v<br>
git branch -M main <br>
git push origin main<br>
if push not work then use this commmand git push -f origin main <br>
